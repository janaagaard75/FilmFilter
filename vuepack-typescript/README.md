## 特色

- Vue 2
- 最新的 Vuex/Router
- Webpack 2
- Babel 6
- 熱重啟工作區域（Hot Loader）
- 可正常建置
- 支援 SASS、CSS、字體、圖片
- 一個內建的小型 Vue 樣板

## 從這開始

```bash
# clone（複製）或 fork（分歧）這個倉庫
$ cd vuepack
$ npm install

# 編輯檔案然後進行開發工作
$ npm run dev
# 將所有檔案打包轉成靜態檔案作為正式釋出
$ npm run build
```

### 給 Windows 使用者

安裝 `git with unix tools` 在你有所行動之前。
